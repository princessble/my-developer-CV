


# Personal Resume Website

<p>This project is a **static personal resume website** built with **HTML5**. </p> 
<p>It presents my professional profile, skills, experience, awards, and contact details in a clean, structured, and easy-to-navigate format.</p>

The project is designed to be:
- Simple
- Beginner-friendly
- Easy to extend with CSS and JavaScript later
- Suitable for hosting on GitHub Pages or any static hosting platform
